# WoW mods

1. Download [Aslain's modpack](https://aslain.com/index.php?/topic/2020-download-%E2%98%85-world-of-warships-%E2%98%85-modpack/).
2. Run it within steam:
    1. In steam Library go to "Add a game" > "Add a non-steam game"
    2. Select the downloaded exe.
3. Once added, right click the game in steam library and go to "Properties" > Compatibility", and select Proton. "Proton Experimental" worked when run on 2025-09-15.
4. Run the "game" in steam.
5. On setup, select the game location in `Z://` under `~/.local/share/Steam/steamapps/common/WoW`.
